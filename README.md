## 七不衰退法 Principles for preventing decline

### A) 民主法治

1. 经常聚集开会
2. 在会议中上下一心地共同处理大众事务
3. 遵循既有的制度，不轻易更改

### B) 社会风气的精神建设

4. 不侵犯妇女
5. 尊崇照顾老人，借重耆老的经验
6. 勤于维护既有的寺庙
7. 都能拥护供养阿罗汉圣者
